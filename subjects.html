Subject 1
Ever since drones have become more available, cheaper and more performant, they enable us to perform many useful missions. However, these drones might be posing safety and security issues. In the proposed project we'll relate to the situation where a drone is trying to protect a closed area from other drones. In the first phase of the project, we'll try to create an application that enables the drone the return to some base point marked by some static object ( like a balloon). A network of cameras will be responsible for identifying the target invader and will guide the drone to the target point. The project is based on hardware including: drone, cameras (we'll be provided on demand)
Subject 2
The challenge is to understand how this program works, and add to it a new computational-geometry algorithm, such as: finding a shortest hamiltonian path, finding a largest blocked circle, dividing a polygon to equi-area parts, etc. Advantages:
  • Contributing to a beautiful software with many worldwide users.
  • Cooperating with many worldwide software developers.
  • Contributing to an open-source project greatly improves your CV.
    • Challenges:
  • Downloading, compiling and running such a large software-system is itself a project.
  • Understanding how it works is another project. So, even a single addition could be considered a great achievement.
